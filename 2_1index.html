Learning GIT

GIT Commands

  • git -v => to check git version
  • git config --global user.name => to know user name
  • git config --global user.email => to know user email
  • git config --global user.email "snehaRoy90737@gmail.com" => to set user email
  • git config --global user.name "Sneha Roy" => to set user name
  • pwd => show the storage location
  • ls => show list of all the folder present in it
  • cd => to go any of these folder present in this above list
  • cd ../ => to go to the previous folder
  • mkdir nameOfFolder => to make a new folder
  • mv nameOfFolder/ newFolder => to rename a folder
  • touch index.html style.css script.js => to create new files all at once
  • mv index.html style.css script.js ./diffFolderName => to move files in different folder
  • git init => to create Git repository
  • git add 2_1index.html 2_1style.css => to move those particular files from "Changes" to Staged Changes
  • git add . => to move all files from "Changes" to "Staged Changes"
  • git status => to check which file is at "Changes" which are at "Staged Changes"
  • git status -s => same as previous command (git status) but information will be in short
  • git commit -m "initial commit" => to commit the file with a massage "initial commit"
  • git log => to know commit history details
  • git checkout idOfTheCommit => to go this particular commit's version
  • git checkout main => to come back to the current version
  • git branch => to see all the existed branch
  • git branch newBranchName => to create a new branch
  • git checkout newBranchName => to go to new branch "newBranchName"
  • git log --all => to show commit history details for all branches
  • git log --all --oneline => to show commit history details for all branches in SHORT
  • git log --all --oneline --graph => same as git log --all --oneline but it also add the branch details
  • git branch --delete branchName => to delete a branch which don't have any commit
  • git branch --D branchName => to delete a branch with commit files
  • git merge branchName => to merge a branch with another
  • git merge branchName -m "commit" => to merge a branch with another with a commit
  • git push => to update code at github
  • git pull => to pull code from remote(git hub) to local(here in vs code in my laptop)
